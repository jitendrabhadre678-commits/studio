# **App Name**: GameFlashX

## Core Features:

- User Account & Authentication: Secure user registration, login, and profile management to track engagement and rewards.
- Interactive Gift Card Explorer: Browse and search digital gift cards across various categories (Shopping, Gaming, Digital, Entertainment, Food, Transport, Payments) with a glass-style search bar and real-time suggestions.
- Reward Unlock Progression: An engaging, full-screen modal with animated progression steps to guide users through the reward verification and readiness process, concluding with a redirection to the external reward page.
- Live Reward Activity Feed: Real-time, interactive pop-up notifications showcasing recent user reward unlocks from around the globe, fostering a sense of community and excitement.
- Creator Program Showcase: Dedicated section detailing the 'GameFlashX Creator Program' with earning structures, eligible platforms, and a clear call-to-action linking to the external signup platform.
- SEO-Optimized Landing Pages: Dedicated, dynamically generated landing pages for popular gift card brands (e.g., /amazon-gift-card, /steam-gift-card) to improve search engine visibility.
- AI Gift Card Description Tool: An AI tool that generates unique, compelling marketing descriptions for new gift card offerings, tailored to the target audience and platform's brand voice.

## Style Guidelines:

- Background: A dynamic linear gradient from vibrant crimson to deep black (135deg, #df104e to #000000), suggesting futuristic depth and energy, complemented by a base dark hue #1c1718 for consistency.
- Primary Color: A bold and captivating red-pink, #df104e, selected to evoke excitement and drive conversions, ensuring high contrast against the dark background for CTAs and critical elements.
- Accent Color: A rich magenta, #b3114c, chosen to be analogous to the primary color, providing depth and distinction for secondary interactive elements or highlighted content.
- Text Color: A soft, understated off-white, #b7a7b1, providing readability and a premium feel against the dark aesthetic, preventing eye strain while maintaining brand cohesion.
- Headline Font: 'League Spartan' (sans-serif), for its strong, modern, and commanding presence, aligning with a tech startup aesthetic. Note: currently only Google Fonts are supported.
- Body Font: 'Garet' (sans-serif), offering a contemporary and clean look for optimal readability in longer text blocks and UI elements. Note: currently only Google Fonts are supported.
- Accent/Decorative Font: 'Anyaran', used sparingly for distinctive branding elements or short, impactful text, contributing to the premium, unique design. Note: currently only Google Fonts are supported.
- Modern, sleek vector icons with a futuristic, gaming-inspired aesthetic, potentially incorporating subtle glow or glassmorphism effects to align with the premium UI design. Logo will feature a lightning bolt integrated with an 'X'.
- Responsive, grid-based layouts prioritizing mobile-first accessibility. Content blocks feature premium 'glass cards' with blur effects for a sophisticated, futuristic look, maintaining ample spacing for clarity.
- Subtle and smooth transitions between states, incorporating glow hover animations for interactive elements, gradient border animations, and a festive confetti animation upon successful reward unlocks, enhancing user engagement.
